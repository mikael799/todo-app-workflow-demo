# Glossary

## Todo

A single unit of work tracked by the app.

- **id** — unique identifier (string)
- **title** — what needs doing (string)
- **completed** — whether the todo is done (boolean)
- **createdAt** — when it was created (timestamp)

## Filter State

A view mode for the todo list.

- **all** — show every todo
- **active** — show only incomplete todos
- **completed** — show only done todos
